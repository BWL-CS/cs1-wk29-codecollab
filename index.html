Variables - A storage location paired with an associated "identifier" (the variable's name or label 🏷️), which contains some known or unknown quantity of information (referred to as a "value"). DECLARING a variable involves using a special keyword, either let or const. ASSIGNING a value to a variable is done with a single equals sign ( = ). Variables are like boxes 📦 that store data 🧠

Statements - Instructions/commands 👉 that are executed by the web browser. A JavaScript statement can be a simple or complex operation that performs an action, such as assigning a value to a variable or calling a function. Statements always end with a semicolon ( ; ) like a stop sign 🛑, whereas "code blocks" (functions, loops, conditionals) open up curly brackets ( { } ) like a section.

Functions - Reusable blocks of code that execute specific tasks (instructions) when called (activated). Functions can take inputs (known as "parameters") and can provide outputs (known as "return values") that can be stored as variables. Functions are like recipes 📝, parameters are like ingredients 🥣, and return values are like a resulting meal 🍲.

Loops (for, while) - Loops allow for a block of code instructions to be repeated 🔁. Conditionals (if, else) - Conditional blocks are used to perform different actions based on different conditions. Conditional expressions are like questions 🤨 that have a answer of either "true" or "false".
